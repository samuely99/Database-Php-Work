This html script shows how to generate results for simple queries.

The following button finds the student(s) with the highest GPA.

The associated SQL code is the following: Select S.Name, S.Id From Student S Where S.GPA >= (Select Max(S1.GPA) From Student S1)



The following button finds the student(s) with the lowest GPA.

The associated SQL code is the following: Select S.Name, S.Id From Student S Where S.GPA <= (Select Min(S1.GPA) From Student S1)
